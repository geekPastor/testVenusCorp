# 📱 Test Technique – Venus Corporation SARL

Ce projet a été réalisé dans le cadre du **processus de recrutement** chez *Venus Corporation SARL*.  
Il s’agit d’une application Android développée en **Kotlin avec Jetpack Compose**.

---

## 🚀 Fonctionnalités

- ✅ Affichage d’une **liste de produits** (nom, description, image)  
- ✅ Fonction de **recherche** pour filtrer les produits en temps réel  
- ✅ Utilisation d’**images locales** avec `painterResource`  
- ✅ Interface moderne grâce à **Jetpack Compose**  

---

## 🛠️ Stack Technique

- **Langage :** Kotlin  
- **UI Toolkit :** Jetpack Compose  
- **Architecture :** MVVM (Model-View-ViewModel)  
- **Outils :**  
  - Android Studio  
  - Gradle  

---

## 📂 Structure du projet

- `ui/sreens` → Composables pour l’interface utilisateur
-  `ui/components`  → Composant utiliser pour affiches des elements d'interface utilisateur 
- `model/` → Classe `ProductData` représentant un produit
